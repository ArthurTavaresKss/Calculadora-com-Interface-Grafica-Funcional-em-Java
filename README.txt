====================PROJECT=JAVA=CALCULATOR====================

===============================================================

==== PT/BR ====
Desenvolvi um projeto simples de calculadora rudimentar fora da 
grade curricular, para aprimorar minha experiência no java e me 
divertir. Utilizei Swing para fazer a interface e um framework 
chamado exp4j para fazer as operações de forma mais resumida. 
Após isso criei um arquivo jar para poder criar um atalho e 
utilizar a calculadora fora da API. Gostei muito de desenvolver 
esse projeto, aprendi muito sobre swing e utilizar lógica para 
resolver operações matemáticas, converter strings, etc.

==== ENG/US ====
I developed a simple, rudimentary calculator project outside the 
curriculum, to improve my Java experience and have 
fun. I used Swing to create the interface and a framework 
called exp4j to perform the operations in a more summarized way. 
After that, I created a jar file so I could create a shortcut and 
use the calculator outside the API. I really enjoyed developing 
this project, I learned a lot about Swing and using logic to 
perform mathematical operations, convert strings, etc.
